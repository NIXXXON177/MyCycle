# Changelog

Все заметные изменения Florea (ранее MyCycle) документируются в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/).

## [1.8.1] — 2026-06-23

### Ребрендинг и надёжность

- Переименование приложения **MyCycle → Florea** (`com.florea.app`)
- **Backup v2**, PIN hash, сброс биометрии после Restore
- CI: релиз `Florea.apk` + `manifest.json` для авто-обновления

## [1.7.5] — 2026-06-19

### Reliability & Data Protection

- **Backup v2**: экспорт в ZIP (`manifest.json`, `settings.json`, `mycycle.db`, `diary_images/`)
- Безопасное восстановление с откатом при ошибке
- Обратная совместимость со старым форматом `.db` (только база)
- Восстановление настроек и PIN (хэш SHA-256 в `settings.json`, без plaintext)
- Биометрия после Restore отключается; `devicePlatform` в `manifest.json`
- Пересоздание путей к фотографиям после переустановки
- Заглушка для отсутствующих фото (`MissingImagePlaceholder`)
- Режим разработчика: генератор stress-тест данных
- Автотесты: backup/restore, миграции v1→v5, обработка нехватки места

## [1.7.0] — 2026-06-19

- Индекс самочувствия 0–100 с уровнями и графиком
- Закономерности с процентами, лучшие/сложные дни цикла
- Карточки на главной: индекс сегодня и «Твой цикл в цифрах»
